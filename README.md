PostRestService
PostRestService — это учебный RESTful сервис, демонстрирующий реализацию CRUD-операций и взаимодействие с базой данных PostgreSQL. Проект создан в образовательных целях для изучения принципов построения бэкенд-приложений, работы с HTTP-протоколом и архитектуры клиент-сервер.

🛠 Технологии
Язык: Java

Фреймворк: Spring Boot

База данных: PostgreSQL

Сборка: Maven

⚙️ Функциональность
Создание, чтение, обновление и удаление записей (CRUD)

Обработка ошибок и понятные HTTP-статусы

Подключение к PostgreSQL через пул соединений
